COMMON ISSUES AND SOLUTIONS
═══════════════════════════════════════════════════════════════════════════════

ISSUE: Code won't build
SOLUTION: 
  1. Check all tools installed: gcc, flex, bison, make
  2. Run: make clean && make
  3. Check for file permissions
  4. Try: make -B (force rebuild all files)

ISSUE: Examples don't run
SOLUTION:
  1. Verify dfa_converter.exe exists
  2. Verify tests/dfa_examples.txt exists
  3. Try: ./dfa_converter < tests/test_input.dfa
  4. Check file permissions

ISSUE: Memory leak detected
SOLUTION:
  1. Run: make clean && make
  2. Ensure using optimized code (dynamic allocation)
  3. Check free_nfa() is complete
  4. Run: valgrind --leak-check=full ./dfa_converter

ISSUE: Word document formatting messed up
SOLUTION:
  1. Select all text (Ctrl+A)
  2. Clear formatting (Ctrl+Space)
  3. Reapply styles
  4. Use Paste Special (Ctrl+Shift+V) for plain text
  5. Reformat code sections as needed

ISSUE: Can't find files
SOLUTION:
  1. Navigate to: C:\Users\legam\Documents\btech\assignment\toc
  2. List files: dir /b
  3. Verify all files are present
  4. Check file permissions
